->Project Name:
  _KeenKeeper.

->Deccription:
  _KeenKeeper is a simple and beautiful web application to help you keep track of meaningful relationships in your life. It allows you to organize friends, monitor how long it has been since you last contacted them, and stay connected with the people who matter most.

->Technologies used:
-  React.js
-  Vite
-  Tailwind CSS
-  DaisyUI
-  React Toastify
-  JSON data (local mock data)

->Features:
-  View and manage your friends list
-  Dashboard showing relationship statistics
-  Track days since last contact
